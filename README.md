# Albo dei Donatori - Museo MEdA di Aquilonia

Web app responsive per la consultazione pubblica dei donatori del Museo MEdA.

L'app si trova in `outputs/donor-wall`.

## Nota sicurezza

Il token Airtable non deve essere pubblicato su GitHub. La configurazione pubblica non contiene credenziali.

Nel passaggio successivo su Render, il token va inserito come variabile privata del servizio.
